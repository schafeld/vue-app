# vue-app

Demo app in Vue (Vue3, SpaBase, CRUD, Pinia, Tailwind,...)

## Repo

https://github.com/schafeld/vue-app

Work in progress...

## Project setup

```bash
npm create vue@latest
cd vue-app
npm install
npm run dev
```

Checking for updates.

```bash
npm install -g npm-check-updates
ncu -u
npm install
```
